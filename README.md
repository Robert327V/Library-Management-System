A Program using dynamic memory to make a linked list of patron objects with functionality to read from file, add to list, remove from list, print list, search for an entry, and edit an entry.

When prompted, input the file you wish to import. If completed correctly, the imported list will be printed and the main menu will open. If not imported correctly, simply type read from the menu, and try again or make the list manually. In the main menu there is an add, remove, read, print, search, and exit option. Simply type in the option you would like to use. Remove is done by searching for an entry using a patron's ID number. To edit an entry, first search using the patron's ID number, answer the prompt, and type in the data type you would like to edit. When completed with the program, return to the main menu and type exit.

Warning!
--------------------
Duplicate ID numbers are not allowed in order to ensure data safety in the remove and edit functions.
